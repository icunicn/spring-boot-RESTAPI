Welcome to MVC Demo!

Ini adalah aplikasi demo yang menggunakan arsitektur MVC (Model-View-Controller) dengan Spring Boot dan Thymeleaf template engine.

Aplikasi ini mendemonstrasikan penggunaan struktur MVC yang proper dengan separation of concerns yang jelas antara Controller, Service, Repository, dan View layers.

REST API Endpoints:
  • GET /api/users - Get all users
  • GET /api/users/{id} - Get user by ID
  • POST /api/users - Create new user
  • PUT /api/users/{id} - Update user
  • DELETE /api/users/{id} - Delete user
Web Interface:
  • GET / - Home page
  • GET /users - Users list page
  • GET /users/new - Add user page
  • GET /users/{id} - User detail page
  • GET /users/{id}/edit - Edit user page
